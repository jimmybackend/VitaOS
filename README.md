# VitaOS
VitaOS con IA es un sistema operativo desde cero, live/RAM, con IA residente en el kernel, auditoría obligatoria en SQLite y nodos cooperativos para emergencia, rescate y operación guiada. Detecta hardware, propone acciones al usuario y registra cada decisión para trazabilidad, análisis posterior y trabajo entre dispositivos conectados en red real.
